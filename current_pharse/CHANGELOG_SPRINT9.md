# Changelog -- Sprint 9 (10--11 апреля 2026)

## PR #11: feat(verdandi): Sprint 9 -- L2 click filters, column-level dimming, UI enhancements

---

### Новые компоненты

#### Profile Modal (11 файлов, ~1,400 LOC)

- **ProfileModal.tsx** -- модальное окно профиля пользователя с 10 вкладками, разделёнными на 3 секции (Account, Interface, System). Управляет маршрутизацией между вкладками и состоянием модала.
- **ProfileTabProfile** -- отображение и редактирование основной информации профиля (имя, email, аватар, роль).
- **ProfileTabSecurity** -- настройки безопасности: смена пароля, двухфакторная аутентификация, активные сессии.
- **ProfileTabAccess** -- управление доступом: роли, группы, назначенные права.
- **ProfileTabAppearance** -- настройки внешнего вида: тема (тёмная/светлая), плотность интерфейса, язык.
- **ProfileTabGraph** -- конфигурация графа: направление layout, анимации, фильтры по умолчанию.
- **ProfileTabActivity** -- журнал активности пользователя: последние действия, история входов.
- **ProfileTabNotifications** -- управление уведомлениями: каналы, частота, типы событий.
- **ProfileTabFavorites** -- избранные элементы: сохранённые запросы, закладки на узлы графа.
- **ProfileTabShortcuts** -- справочник горячих клавиш с возможностью переназначения.
- **ProfileTabTokens** -- управление API-токенами: создание, отзыв, срок действия.

#### Command Palette (1 файл, 312 LOC)

- Вызов через Ctrl+K / Cmd+K.
- 5 встроенных команд.
- Интеграция с useSearch для поиска по командам.
- Полная клавиатурная навигация (стрелки, Enter, Escape).

#### Search Palette (1 файл, 643 LOC)

- Вызов по клавише `/`.
- 7 фильтров по типу элемента.
- История недавних запросов.
- Секция скрытых узлов с возможностью раскрытия.
- Правила маршрутизации для навигации к найденным элементам.

#### Inspector Components (3 файла, ~160 LOC)

- **InspectorColumn** -- панель инспектора для отдельного столбца: тип данных, lineage, трансформации.
- **InspectorJoin** -- панель инспектора для join-связей: тип join, ключи, условия.
- **InspectorParameter** -- панель инспектора для параметров: имя, тип, значение по умолчанию.

### Новые хуки (2 файла, ~147 LOC)

- **useHotkeys** -- универсальный хук для регистрации горячих клавиш с поддержкой модификаторов и контекстных областей.
- **useSearchHistory** -- хук для управления историей поисковых запросов: сохранение, загрузка, очистка.

---

### Новые store slices (2 файла, ~207 LOC)

- **undoSlice** -- реализация undo/redo для состояния Loom: стек операций, откат, повтор.
- **persistSlice** -- персистентность состояния: сохранение в localStorage, восстановление при загрузке.

### Расширенные компоненты

- **InspectorRoutine** -- добавлена поддержка отображения параметров и расширенных метаданных.
- **InspectorStatement** -- улучшено отображение SQL-выражений и связанных столбцов.
- **Header (3-norns)** -- интеграция трёх элементов управления (поиск, команды, профиль) в шапку приложения.
- **FilterToolbar** -- добавлены L2 click-фильтры с column-level dimming и кросс-фильтрация.
- **ToolbarPrimitives** -- новые примитивы для кнопок фильтрации и индикаторов состояния.

---

### Исправления багов

- **Column dropdown wipe** -- исправлена ошибка, при которой выпадающий список столбцов очищался при переключении между узлами.
- **Filter cross-clearing** -- устранена проблема, когда сброс одного фильтра некорректно сбрасывал смежные фильтры.
- **Edge CSS visibility** -- исправлена видимость рёбер графа при определённых комбинациях фильтров.
- **console.warn hack removed** -- удалён временный хак подавления предупреждений в консоли.
